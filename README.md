# OnlineFoodOrdering_Group4
A project on online food ordering created by group 4 members. Group 4 backend project at KSTU , Computer Science Department.


Project structure
foodorder/
├── index.html              ← Homepage (public landing page)
├── register.php            ← User registration (with profile photo upload)
├── login.php               ← User sign-in
├── dashboard.php           ← Main app (menu, cart, orders, profile)
├── logout.php              ← Session destroy + redirect
├── profiles/
│   └──  pictures           ← User profile pictures (auto-created)
└── Uploads/
    └──  documents          ← User accounts + cart + orders (auto-created)
